# Top of Mind

Top of mind  is a minimal static site template that's meant to be
quick for local search. 

Update the `src/data.json` with your data.

```json
[
  {
    "title": "Star Wars: A New Hope",
    "url": "http://www.imdb.com/title/tt0076759/",
    "tags": [
      "IMDB"
    ]
  }
  //... more object...
]
```
